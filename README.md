# IBGFelps '-'

Esse repositório contém os scripts (em R), principais resultados e relatório final da análise descritiva dos dados do IBGFelps 2026.

A análise é dividida em 3 etapas:

1. Análise cruzada dos dados, pra responder perguntas como "Como cada faixa etária monta seu hambúrguer, em média?" ou "As pessoas mais 'sim' estão mais concentradas em alguma região do Brasil?", etc.

2. Análise comparativa dos dados, usando as bases do Censo IBGE 2022 (mais recente) e da Pesquisa Nacional por Amostra de Domicílios Contínua (PNAD-C) de 2025

3. Análise inferencial de correlações e predição (?)
